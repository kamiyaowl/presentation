本プレゼン資料生成には[backslide](https://github.com/sinedied/backslide) を使用しており、`/template`と`package.json`を準備しただけになります。

ですので、以下の手順でプレゼンテーション作成環境を整えることができます。

1. 本リポジトリをfork
1. `$ npm install`
1. `$ npm run reset`
1. `$ npm run build`
